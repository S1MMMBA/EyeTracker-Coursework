Eye Tracker Program
Программа для трекинга взгляда и генерации криптографических ключей на основе движений глаз.

   Основные возможности
Трекинг взгляда в реальном времени с использованием MediaPipe

Детекция движений глаз с фильтрацией шума

Генерация энтропии из биометрических данных

Создание криптографических ключей (128/256 бит)

Визуализация процесса с отображением статистики

Сохранение результатов в различные форматы

  Быстрый запуск
Предварительные требования
Python 3.8 или выше

Веб-камера

4 ГБ оперативной памяти

Интернет-соединение (для загрузки моделей MediaPipe при первом запуске)

Установка
Клонируйте репозиторий:

bash
git clone https://github.com/ваш-username/EyeTracker-Coursework.git
cd EyeTracker-Coursework/code
Установите зависимости:

bash
pip install -r requirements.txt
Запуск программы
bash
python src/EyeTracker.py
Управление программой
После запуска используйте следующие клавиши:

Клавиша	Действие	Описание
Q или ESC	Выход	Завершение работы программы
S	Статистика	Показать статистику в консоли
B	Генерация битов	Сгенерировать 128 случайных битов
K	Генерация ключа	Сгенерировать 256-битный криптографический ключ
R	Сброс	Сбросить все накопленные данные
 Интерфейс программы
Программа отображает в реальном времени:

Видеопоток с веб-камеры

Ключевые точки глаз (зеленые/оранжевые точки)

Точку взгляда (красная точка при движении, оранжевая при покое)

Статистическую информацию в левом верхнем углу
