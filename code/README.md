# 👁 EyeTracker — Генератор криптографических ключей

Генератор криптографических ключей на основе биометрических данных: движений глаз и голоса пользователя. Использует аппаратную случайность из физиологически непредсказуемых источников.

---

## 📋 Требования

- **Python** 3.10-3.12. Python 3.13 не подходит для текущей версии
  проекта: он ставит MediaPipe без legacy API `mp.solutions.face_mesh`.
- **Веб-камера** (для режимов «Взгляд» и «Оба»)
- **Микрофон** (для режимов «Голос» и «Оба»)
- **OS:** Windows 10/11, Linux, macOS

---

## 📦 Установка зависимостей

### 1. Клонировать репозиторий

```bash
git clone <url-репозитория>
cd EyeTracker-Coursework/code
```

### 2. (Рекомендуется) Создать виртуальное окружение

```bash
# Если используется pyenv/pyenv-win, версия уже зафиксирована
# в файле .python-version. При необходимости установите ее:
pyenv install 3.12.7

# Убедитесь, что команда показывает Python 3.12.x
python --version

python -m venv venv
```

Если `python --version` показывает Python 3.13, не продолжайте установку
зависимостей: сначала установите Python 3.12 и создайте окружение его
интерпретатором.

Активация:
- **Windows:**
  ```bash
  venv\Scripts\activate
  ```
- **Linux / macOS:**
  ```bash
  source venv/bin/activate
  ```

### 3. Установить все зависимости

```bash
python -m pip install -r requirements.txt
```

Или по одному, если нужна отладка:

```bash
pip install "opencv-contrib-python<4.12"  # захват и обработка видео
pip install "numpy<2"                     # математические операции
pip install mediapipe==0.10.21            # трекинг лица и глаз
pip install PyQt5                         # графический интерфейс
pip install sounddevice                   # захват звука с микрофона
pip install matplotlib                    # визуализация (используется ядром)
```

## 🖥 Интерфейс и режимы работы

### Выбор источника энтропии

В левой панели переключатель **«Источники энтропии»** имеет три положения:

| Положение | Описание |
|-----------|----------|
| 👁 Взгляд | Только камера; собираются движения глаз |
| 👁+🎤 Оба | Камера + микрофон (максимальная энтропия) |
| 🎤 Голос | Только микрофон; камера не запускается |

---

## 📖 Пошаговое использование

### Шаг 1 — Выбрать режим
Переключите тумблер на нужный источник энтропии **до** нажатия «Начать трекинг».

### Шаг 2 — Запустить трекинг
Нажмите кнопку **▶ Начать трекинг**.
- В режиме с камерой: дождитесь появления видеопотока.
- В режиме с микрофоном: убедитесь, что индикатор микрофона стал зелёным.

### Шаг 3 — Накопить энтропию

**Взгляд:** следите глазами за движущимися объектами на экране. Для ускорения:
- Нажмите **▶ YouTube** или **🌐 Браузер** — откроется видео/сайт как источник стимулов.
- Включите **«Случайные триггеры»** — на экране появятся звёзды-мишени.

Минимум для генерации ключа: **20 движений** (следите за счётчиком в правой панели).

**Голос:** говорите или воспроизводите звук рядом с микрофоном. Следите за виджетом аудиовизуализации и ростом показателя «Качество сигнала».

### Шаг 4 — Сгенерировать ключ
Нажмите **128 бит** или **256 бит** в секции «Генерация ключа».

Ключ автоматически сохранится в файл `eye_voice_key_<bits>_<дата>.bin` в директории запуска.

### Шаг 5 — Сохранить / скопировать
- **📋 Копировать** — скопировать hex-строку ключа в буфер обмена.
- **💾 Сохранить ключ** — сохранить в произвольное место.
- **⬇ Скачать пул энтропии (.bin)** — сохранить сырой пул, из которого
  был получен последний сгенерированный ключ.

## ⚠️ Возможные проблемы

### Камера не открывается
- Проверьте, что камера не занята другим приложением.
- Попробуйте переключить индекс камеры в `CameraThread.run()` (`cv2.VideoCapture(0)` → `cv2.VideoCapture(1)`).

### MediaPipe не установлен
```bash
python -m pip install -r requirements.txt
```

Если используется Python 3.13, `pip install mediapipe` поставит новую
ветку MediaPipe без `mp.solutions.face_mesh`. Для этого проекта нужно
пересоздать окружение под Python 3.10-3.12:

```powershell
deactivate
Remove-Item -Recurse -Force venv
pyenv install 3.12.7
python -m venv venv
venv\Scripts\activate
python -m pip install -r requirements.txt
```

### Микрофон не работает
```bash
pip install sounddevice
```
На Linux может потребоваться:
```bash
sudo apt install libportaudio2
```

### Ошибка PyQt5 на Linux
```bash
sudo apt install python3-pyqt5
# или через pip:
pip install PyQt5
```

### «Недостаточно данных» при генерации ключа
- В режиме «Взгляд»: нужно минимум **20 зафиксированных движений** — продолжайте двигать глазами.
- В режимах «Голос» / «Оба»: нужен пул минимального размера (60 байт для 128-бит, 117 байт для 256-бит) — продолжайте говорить.

## 🔐 Безопасность

Генерируемые ключи основаны на физически непредсказуемых биометрических данных и проходят хэширование через **SHA-256**. Параметры минимального пула энтропии рассчитаны по методологии **NIST SP800-90B** с коэффициентом запаса ×3.

Каждая генерация использует только новые данные, накопленные после
предыдущего ключа. Использованный диапазон пула помечается как
израсходованный, поэтому следующий ключ требует нового движения глаз
и/или нового голосового сегмента.

Сохраняйте сгенерированные `.bin`-файлы в надёжном месте — повторно сгенерировать тот же ключ невозможно.
